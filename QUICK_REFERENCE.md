## 🔑 Quick Actions

### For Customers

| Action | Where to Click |
|--------|---------------|
| Create Account | Homepage → Register |
| Login | Homepage → Login |
| Book Event | Dashboard → Select Event → Book Now |
| View Bookings | Sidebar → My Bookings |
| Send Message | Sidebar → Messages |
| Check Notifications | Bell Icon (top right) |
| Update Profile | Avatar → Profile |
| Make Payment | My Bookings → Select Booking → Pay |
| Cancel Booking | My Bookings → Select Booking → Cancel |

### For Admins

| Action | Where to Click |
|--------|---------------|
| View Dashboard | Sidebar → Dashboard |
| Manage Bookings | Sidebar → Bookings |
| Walk-in Booking | Sidebar → Walk-in Booking |
| View Calendar | Sidebar → Calendar |
| Manage Inventory | Sidebar → Inventory |
| View Payments | Sidebar → Payments |
| Manage Users | Sidebar → Users |
| View Reports | Sidebar → Reports |

## 📊 Booking Status Guide

| Status | Icon | Meaning |
|--------|------|---------|
| Pending | 🟡 | Awaiting admin review |
| Confirmed | 🔵 | Booking approved |
| Pending Payment | 🟠 | Payment required |
| Partial Payment | 🟢 | Down payment received |
| In Design | 🎨 | Design in progress |
| Completed | ✅ | Event finished |
| Cancelled | 🔴 | Booking cancelled |
| Rejected | ❌ | Booking declined |

## ⌨️ Localhost Quick Commands

```bash
# Start server
php artisan serve

# Start frontend (keep running)
npm run dev

# Clear all caches
php artisan optimize:clear

# Run migrations
php artisan migrate
```
